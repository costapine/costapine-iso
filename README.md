# Costapine ISO

ISO generation configs and live boot initramfs for Costapine Linux.

Part of the [Costapine Linux](https://github.com/costapine/costapine) project.

## Structure

- `grub/` — GRUB bootloader configuration (UEFI)
- `syslinux/` — Syslinux/isolinux configuration (BIOS)
- `initramfs/` — Custom init script for live boot
